## 今日知识点（2026-05-13）
1.  **String 字符串**
    - 是不可变对象，每次修改都会生成新对象
    - `==` 比较内存地址，`equals()` 比较内容
    - 判空优先用 `str != null && !str.isEmpty()`，避免空指针异常

2.  **运算符优先级口诀**
    括号第一高，单目紧跟着；
    先乘除后加减，再比较再逻辑；
    && 在前 || 在后，三元居中，赋值最后。
