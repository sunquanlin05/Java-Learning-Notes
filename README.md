## 2026-05-13
1.**String 字符串**
    - 是不可变对象，每次修改都会生成新对象
    - ==比较内存地址，equals()比较内容
    - 判空优先用 `str != null && !str.isEmpty()`，避免空指针异常
2.**for循环：fori+回车，快速生成for循环**
   实际开发中，推荐把循环变量定义在for循环内部，避免变量名冲突的问题。
3.**while循环**
    初始化语句;
    while()
    {
        循环体语句（被重复执行的代码）;
        迭代语句;
    }
//知道循环几次——for循环，不知道循环几次建议使用while循环，但在功能上二者无区别，能用for循环实现的也能用while实现，反之亦然
4.**do-while循环**
  初始化语句;
  do{
      循环体语句;
      迭代条件;
  }while(循环条件);//特点：先执行一次再判断条件，因此至少执行一次，比如抢票，开始是没抢到票的，抢到票了才能显示自己有票，或者是点餐，刚开始是没点的，点完了才能显示已经点了什么菜
5.**死循环**
  for(;;){
      ...
  }
  while(true){
      ...
  }//这是最经典的死循环写法
  do{
      ...
  }while(true);
  //注意只有上一个死循环注释掉，下一个死循环才能执行
  //服务器运行/接口请求失败时循环重试直到成功或达到最大失败次数都是死循环的体现
6.**输出方式区别**
    print()：同一行打多个东西
    println()：打一行就换行（更常用）
    printf()：要格式化（比如保留小数、对齐）
